The only thing left is CSS...
So here we go.
CSS stands for Cascading Style Sheets. Using CSS allows someone to use similar styles (font color, text size, backgrounds...) on mulitple pages, without having to copy pasta the same section of code over 9000 times. Using CSS also allows someone to easily change their entire website by simply modifying one CSS class file. You can even link multiple files on the same page and specify different mediums for which they'll display, allowing your mom's Iphone and your brother's PSP to use a different style sheet and layout!
The primary reason CSS is "cascading" is because of the fact mentioned above, being able to link multiple sheets to the same document. Every page of this site uses 2-3 CSS files (1 being the printer page, which can be seen in the 'print this page' section of the 'Form' page).
