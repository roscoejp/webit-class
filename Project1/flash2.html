Flash on splash.


This flash is a little more... complicated than that other, simple animation.

Instead of using a traditional flash button, to manipulate the robot, I used something called a 'mask'. It doesn't work like a traditional mask, however, because the mask coveres the entire page, and then I had to determine what became unmasked and when. I used a movie clip behind the mask and a motion tween inside the movie clip to move the robot. I used the movie clip because I needed to 'seek' to certain positions of the movie whenever a user hovered over the clip and moved away from the clip. I haven't quite got the actionscript where I want it, and in the end I hope to have the robot reverse it's motion tween mid-way through said tween. I haven't done much javascripting though, so actionscript is all new to me...

Song: The Becoming
Artist: Nine Inch Nails

import flash.events.MouseEvent;
import flash.net.navigateToURL;

btn_mc1.buttonMode = true;
btn_mc1.addEventListener(MouseEvent.ROLL_OVER, onButtonOver);
btn_mc1.addEventListener(MouseEvent.ROLL_OUT, onButtonOut);
btn_mc1.addEventListener(MouseEvent.CLICK, onButtonClick);

function onButtonOver(e:MouseEvent): void
{
e.currentTarget.gotoAndPlay("over");
}

function onButtonOut(e:MouseEvent): void
{
e.currentTarget.gotoAndPlay("out");
}

function onButtonClick(e:MouseEvent): void
{
var url:String = "index.html";
var request:URLRequest = new URLRequest(url);
try {
navigateToURL(request, '_self'); // second argument is target
}
catch (e:Error) { trace("Error occurred!"); };
}
